# AstrBot Plugin: TodoPal

AI 驱动的智能待办事项管理插件。只需通过自然语言描述，即可自动解析时间、日期和任务内容。

## 功能特性

- **无前缀自然语言交互 (Smart Agent Mode)**：无需死记硬背指令前缀，直接用自然语言交流（如：“帮我记一下买菜”、“把第一条做完了”）。
- **智能意图识别**：通过 LLM 自动理解你是要添加、完成、修改还是查看待办。
- **主动定时提醒与总结**：支持在指定时间段内（如 09:00 - 22:00）循环提醒未完成的待办，并在每天晚间（如 23:00）发送总结报告。
- **跨日自动顺延**：今天没做完的事情，明天自动顺延，绝不遗漏。
- **Persona 人格支持**：主动提醒时支持套用 AstrBot 预设的人格，提供情绪价值。

## 配置指南

你可以在 AstrBot 的 WebUI 管理面板中对 TodoPal 进行丰富的可视化配置：

1. **自定义特征词**：添加你常用的口头禅作为唤醒词（如“记”、“待办”、“做完”）。
2. **主动提醒设置**：
   - 是否开启提醒/总结
   - 提醒时间间隔（例如每 2 小时）
   - 提醒时段（避免打扰休息）
3. **人格设定**：选择主动发消息时使用的语气人格。

## 使用方法

你可以使用**纯自然语言**（只要包含你在配置中设置的特征词即可），或者使用以下**快捷指令**：

### 1. `todo`：新建/覆盖待办清单
使用 `todo` 开头，表示声明一个**全新**的待办清单。如果当天已有记录，使用该命令并在确认后，会**直接覆盖**原有记录。

**示例：**
> 用户：`todo 明天下午3点给谭老师发邮件，晚上8点健身`
> 
> Bot：
> 【新建/覆盖模式】
> 2026年3月17日 星期二
> 
> 1. 15:00 给谭老师发邮件
> 2. 20:00 健身
> 
> 如果正确，请回复“确认”。

### 2. `add`：追加待办事项
使用 `add` 开头，表示在现有的待办清单上**追加**新的任务，不会影响已有的记录。

**示例：**
> 用户：`add 今天下午再加个会`
> 
> Bot：
> 【追加模式】
> 2026年3月16日 星期一
> 
> 1. 下午再加个会
> 
> 如果正确，请回复“确认”。

### 3. `done`：标记完成
使用 `done` 开头，快速将某一项或多项任务标记为已完成（✅）。支持通过序号或内容片段进行匹配。

**示例 1：按序号完成（支持批量）**
> 用户：`done 1, 2` 或 `done 第1条`
> 
> Bot：
> 太棒了！已更新状态：
> 2026年3月16日 星期一
> 
> 1. ✅ 15:00 给谭老师发邮件
> 2. ✅ 20:00 健身

**示例 2：按内容模糊匹配**
> 用户：`done 健身`
> 
> Bot：
> 太棒了！已更新状态：... (对应项目打上✅)

### 4. `fix`：修改待办
使用 `fix` 开头，修改某一条已存在的待办内容。

**示例：**
> 用户：`fix 2 改成光电数据集会议`
> 
> Bot：
> 已修改第 2 条：
> ...（展示更新后的列表）

### 5. `check`：查看待办
使用 `check` 开头，随时查看当天的待办清单。

**示例：**
> 用户：`check`
> 
> Bot：
> 今日待办清单：
> ...（展示当前列表）

## 数据存储

本插件使用本地 JSON 文件存储数据，无需额外数据库配置。

**文件结构**：
```text
data/plugin_data/todopal/
├── {platform}/           # 平台名称 (如 qq, wechat)
│   └── {user_id}/        # 用户 ID
│       └── {year}/       # 年份
│           └── {month}/  # 月份
│               └── {date}.json  # 每天一个文件
```

**数据格式示例**：
```json
[
  {
    "id": "20260316-abcdef",
    "date": "2026-03-16",
    "time": "15:00",
    "content": "给谭老师发邮件",
    "status": "pending",  // pending: 待办, done: 已完成
    "created_at": "2026-03-16 10:00:00",
    "updated_at": "2026-03-16 10:00:00",
    "done_at": null,
    "source_text": "todo 明天下午3点给谭老师发邮件"
  }
]
```
