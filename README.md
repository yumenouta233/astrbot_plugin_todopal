# AstrBot Plugin: TodoPal

AI 驱动的智能待办事项管理插件。只需通过自然语言描述，即可自动解析时间、日期和任务内容。

## 功能特性

- **自然语言解析**：直接用平时说话的方式描述待办事项，无需复杂格式。
- **智能时间识别**：自动理解“明天下午”、“下周三”、“晚上8点”等时间表达。
- **多任务处理**：一条消息中可以包含多个待办事项。

## 使用方法

### 1. 添加待办事项

直接发送以 `todo` 开头的消息（无需斜杠）。

**示例 1：指定具体时间**
> 用户：`todo 明天下午3点给谭老师发邮件`
> 
> Bot：
> 2026年3月17日 星期二
> 
> 1. 15:00 给谭老师发邮件
> 
> 如果正确，请回复“确认”。

用户回复“确认”后，Bot 会保存数据。如果当天已有数据，Bot 会提示选择“追加”或“覆盖”。

### 2. 标记完成

使用自然语言告诉 Bot 哪件事做完了。

**示例：**
> 用户：`第1个做完了` 或 `发邮件做完了`
> 
> Bot：
> 太棒了！已将“给谭老师发邮件”标记为完成。

## 数据存储

本插件使用本地 JSON 文件存储数据，无需额外数据库配置。

**文件结构**：
```text
data/plugin_data/todopal/
├── {platform}/           # 平台名称 (如 qq, wechat)
│   └── {user_id}/        # 用户 ID
│       └── {year}/       # 年份
│           └── {month}/  # 月份
│               └── {date}.json  # 每天一个文件
```

**数据格式示例**：
```json
[
  {
    "id": "20260316-abcdef",
    "date": "2026-03-16",
    "time": "15:00",
    "content": "给谭老师发邮件",
    "status": "pending",  // pending: 待办, done: 已完成
    "created_at": "2026-03-16 10:00:00",
    "updated_at": "2026-03-16 10:00:00",
    "done_at": null,
    "source_text": "todo 明天下午3点给谭老师发邮件"
  }
]
```
